# Annotatin tool
 web based annotation tools for machine learning.

 ## Prepare
- Install node environment. ( Tested on node version 16.13 )
- exec commands below.
 ```bash
 cd THIS_REPOSITORY
 npm install
 ```

 ## Run
 ```bash
 cd THIS_REPOSITORY
 npm run serve

 ```
 open http://localhost:8080 at Chrome browser


 ## Build
 ```
 cd THIS_REPOSITORY
 npm run build
 ```
